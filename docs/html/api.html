API documentation¶

Node¶

class vntree.Node(name=None, parent=None, data=None, treedict=None)¶

Node is a class for creating vntree nodes.

Parameters:name (str or None) – node name
add_child(node)¶

Add a child node to the current node instance.

Parameters:node (Node) – the child node instance.
Returns:True if successful.
get_node_by_nodepath(nodepath)¶

Get node from nodepath e.g. nodepath=”/rootnode.name/child.name/gchild.name” (absolute path) nodepath=”child.name/gchild.name” (relative path) WARNING: use of this method assumes that sibling nodes have unique names, if this is not the case «get_node_by_coord» can be used instead.

nodepath¶

Return this node’s nodepath in the form: ‘/rootnode.name/childnode.name/grandchildnode.name’ WARNING: use of nodepath assumes that sibling nodes have unique names.

remove_child(node)¶

Remove a child node from the current node instance.

Parameters:node (Node) – the child node to be removed.
Returns:True if successful.
